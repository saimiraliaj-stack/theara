# THÈA RA MODE & DESIGN Website

Statische Multi-Page-Website für GitHub Pages oder normales Hosting.

## Dateien
- `index.html` – Startseite
- `arbeit.html` – Galerie / Unsere Arbeit
- `kontakt.html` – Kontaktseite
- `impressum.html` – Impressum
- `styles.css` – gesamtes Design
- `script.js` – mobiles Menü
- `assets/logo.png` – hier deine finale Logo-Datei einsetzen
- `assets/images/Foto1.jpg` bis `Foto12.jpg` – hier deine Bilder einsetzen

## GitHub Pages Deployment
1. Repository auf GitHub erstellen
2. Alle Dateien aus diesem ZIP hochladen
3. In GitHub auf `Settings` → `Pages`
4. Unter `Build and deployment` bei `Source` → `Deploy from a branch`
5. Branch `main` und Ordner `/ (root)` wählen
6. Speichern

Danach veröffentlicht GitHub die Seite automatisch.
