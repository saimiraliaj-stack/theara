# THÈA RA MODE & DESIGN

GitHub Pages ready static website.

## Structure
- `index.html`
- `arbeit.html`
- `studio.html`
- `kontakt.html`
- `impressum.html`
- `styles.css`
- `script.js`
- `assets/logo.svg`
- `assets/images/Foto1.jpg` ... `Foto12.jpg`

## Replace images
Keep the same filenames inside `assets/images/`:
- Foto1.jpg
- Foto2.jpg
- ...
- Foto12.jpg

## Deploy to GitHub Pages
Push everything to the root of your repository and enable:
- Settings
- Pages
- Deploy from a branch
- Branch: `main`
- Folder: `/root`
